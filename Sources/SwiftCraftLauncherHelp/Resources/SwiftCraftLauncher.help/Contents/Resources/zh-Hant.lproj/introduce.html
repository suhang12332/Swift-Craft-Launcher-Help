專案概覽

Swift Craft Launcher 是一款使用 SwiftUI 構建的原生 macOS Minecraft 啟動器，提供流暢高效的遊戲體驗。專為現代 macOS 系統設計，具有全面的模組載入器支援、Microsoft 帳戶認證和直觀的遊戲管理功能。

基礎功能

  • 多版本 Minecraft 支援 - ARM：1.13+，Intel：未測試

  • Microsoft 帳戶認證 - 安全的 OAuth 整合，支援設備代碼流程

  • 模組載入器支援 - Fabric、Quilt、Forge 和 NeoForge，支援自動安裝

  • 資源管理 - 一鍵安裝模組、資料包、著色器和資源包

用戶體驗

  • 原生 macOS 設計 - 使用 SwiftUI 構建，遵循 Apple 人機介面指南

  • 多語言支援 - 本地化介面，帶國旗指示器

  • 智能路徑管理 - Finder 風格的面包屑導航，自動截斷長路徑

  • 性能優化 - 高效的快取和記憶體管理

高級配置

  • Java 管理 - 每個配置檔案獨立的 Java 路徑配置，支援自動版本檢測

  • 記憶體分配 - 可視化滑塊設置 Xms/Xmx 參數

  • 自定義啟動參數 - 自定義 JVM 和遊戲參數 （即將推出）

系統要求

  • macOS：14.0 或更高版本

  • Java：8 或更高版本（用於 Minecraft 運行時）

安裝

使用 Homebrew Tap（推薦）

brew install --cask suhang12332/swiftcraftlauncher/swift-craft-launcher brew tap suhang12332/swiftcraftlauncher brew install --cask swift-craft-launcher

提示：我們為 SwiftCraft Launcher 創建了專門的 Homebrew Tap。

預編譯版本

從 GitHub Releases 下載最新版本。

【註】所有當前可用版本均為測試版本；穩定版本即將推出。

從源碼構建

  1. 克隆倉庫

    git clone https://github.com/suhang12332/Swift-Craft-Launcher.git cd Swift-Craft-Launcher

  2. 在 Xcode 中打開

    open SwiftCraftLauncher.xcodeproj

  3. 構建並運行 使用 Xcode (⌘R)

構建要求：

  • Xcode 13.0+

  • Swift 5.5+
