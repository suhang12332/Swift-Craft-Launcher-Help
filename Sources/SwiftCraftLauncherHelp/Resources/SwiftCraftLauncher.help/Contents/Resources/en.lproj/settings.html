Launcher Settings

Swift Craft Launcher provides various configuration options to customize your experience.

General Settings

Language

You can set the launcher interface language. Changes will take effect after restarting the launcher.

Appearance

You can set the launcher appearance to light, dark, or follow system. The effects are shown below.

Working Directory

You can set the launcher's working directory. All game files managed by the launcher will be placed in this folder, as shown below.

Important: After changing the working directory, games in the original path will no longer appear in the "Game List". It is recommended to transfer or re-download them. It's best to change the working directory when first opening the launcher.

Download Settings

The number of parallel downloads largely determines your download speed, but higher concurrency also brings greater performance overhead.

If you are not familiar with the meanings of Minecraft Version URL, Modrinth API URL, and Git Proxy URL, please do not modify them. Changing them may cause game resource download failures.

If you accidentally change them, restore them to their default values:

  • Minecraft Version URL: https://launchermeta.mojang.com/mc/game/version_manifest.json

  • Modrinth API URL: https://api.modrinth.com/v2

  • Git Proxy URL: https://ghfast.top

Game Settings

Automatic Dependency Handling

When this option is enabled, any dependencies required for downloading resources will be automatically downloaded without showing notifications. The following page will not appear.

Java Path

You can change this option to select different Java versions to launch the game.

Below are MC versions and their minimum required Java versions:

  • 1.12 - Java 8

  • 1.17 - Java 16

  • 1.18 - Java 17

  • 1.20.5 - Java 21 (64-bit)

Global Memory Allocation

You can control the amount of memory allocated to the game. The minimum value is 512MB. Generally, larger modpacks require more memory.
