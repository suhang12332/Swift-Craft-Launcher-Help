Add Resources

Swift Craft Launcher supports automatically downloading mods, resource packs (or texture packs), shaders, and data packs from CurseForge and Modrinth, and adding them to your game.

Add Mods

Important: Vanilla Minecraft does not support mods. If you want to install mods, make sure you have properly installed a mod loader.

Step 1: Enter Resource Library

In the "Game List" on the left, select the game for which you want to install mods, then click the top button to enter the "Resource Library".

Step 2: Search and Install Mods

Select the "Mods" option at the top, use the search bar on the right to find the mod you want to add. Once you find the mod, click "Install".

Step 3: Handle Dependencies

Some mods may display an interface as shown. This is because the mod requires other mods as dependencies to run. Don't worry, after clicking "Download All Dependencies and Continue", the launcher will automatically install them for you. After the mod installation is complete, click the top button to return to "Installed".

If you don't want to use the automatic dependency installation feature, you can turn it off in the launcher settings.

Step 4: Confirm Installation

When the mod you just downloaded appears on the "Installed" page, the mod has been installed.

Step 5: Verify Mod Loading

At this point, launching the game will show that mods have been successfully loaded.

Add Resource Packs

Step 1: Enter Resource Library

In the "Game List" on the left, select the game for which you want to install resource packs, then click the top button to enter the "Resource Library".

Step 2: Search and Install Resource Packs

Select the "Resource Packs" option at the top, use the search bar on the right to find the resource pack you want to add. Once you find a suitable resource pack, click "Install". After completion, click the top button to return to "Installed".

Step 3: Confirm Installation

When the "Installed" page displays the resource pack you just downloaded, the resource pack has been installed.

Step 4: Load in Game

Launch the game, and you will see that the resource pack has been successfully recognized. Click the small triangle to load the resource pack.

Add Shaders

Prerequisites: If you want to install shaders, make sure you have properly installed one of the following: Optifine, Iris Shader, or Oculus.

Step 1: Enter Resource Library

In the "Game List" on the left, select the game for which you want to install shaders, then click the top button to enter the "Resource Library".

Step 2: Search and Install Shaders

Select the "Shaders" option at the top, use the search bar on the right to find the shader you want to add. Once you find the shader, click "Install". After completion, click the top button to return to "Installed".

Step 3: Confirm Installation

When the "Installed" page displays the shader you just downloaded, the shader has been installed.

Step 4: Load in Game

Launch the game, and you will see that the shader has been successfully recognized. Click the shader you downloaded to load the shader pack.

Add Data Packs

Data packs provide players with more ways to customize Minecraft game content, including but not limited to advancements, recipes, loot tables, enchantments, damage types, mob variants, and world generation.

Note: The add data pack feature is still under development. Please stay tuned for updates.

Manually Add Resources

If you can't find the resources you need in the resource library, you can manually add resources to your game.

Please prepare your resource files first: mods (.jar), resource packs (.zip), data packs (.zip), shader packs (.zip), configuration files (.toml, .json, .yaml, etc.), and save files (folders).

Step 1: Open Game Directory

In the "Game List" on the left, select the game to which you want to add resources, then click the "Path" button at the top.

Step 2: View Game Directory

The launcher will open Finder and automatically navigate to the game's directory, as shown below.

Step 3: Place Resource Files

Place resource files into the corresponding folders:

  • config: Configuration files

  • mods: Mod files

  • resourcespack: Resource pack files

  • saves: Save files

  • shaderpacks: Shader pack files

If you add configuration files or mods, you need to restart the game for them to take effect; other resources don't require a restart.

Add Modpacks

Step 1: Search Modpack

In the "Resource List" on the left, select "Modpacks". In the top search bar, search for the modpack you want to add, then click "Install".

Step 2: Configure Modpack

Select "Game Version" and "Modpack Version", name the version, then click "Download".

Step 3: Wait for Installation to Complete

The launcher will automatically install Java and the modpack for you.

Step 4: Complete Installation

The installed modpack will appear in the "Game List" on the left.
