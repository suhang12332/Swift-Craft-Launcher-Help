Swift Craft Launcher Help

Welcome to Swift Craft Launcher, a native macOS Minecraft launcher built with SwiftUI. This help guide will walk you through all features and how to use them.

Note: This documentation is based on version 0.3.1-beta(31). The launcher and documentation are still in early development, and there may be some inconsistencies. Thank you for your understanding.

  • Account Login - Set up your Minecraft account

  • Add Game - Install Minecraft with mod loaders

  • Add Resources - Install mods, resource packs, and more

  • Launcher Settings - Configure the launcher

  • Edit Skins - Customize your player skin
